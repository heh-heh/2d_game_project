# road map

### 플레이어 스킬
- 아직 설정 정도만 존재 하며 제작 하진 않음

### 몬스터
- 현재 4가지가 존재 하며 실직적으로는 big, speed, 일반 3가지만 출현
- 추후 시간에따라 보스 출현 구현 예정

### 몬스터 스폰 위치 지정
- 몬스터가 스폰은 하나 아직까진 미흡

### 레벨업 시 선택창 ui
- exp 환수만 작동 하는중
- 스킬 선택은 스킬을 일부 제작 후 만들 예정

### esc 누를 시 메뉴창 ui
- ui이 형태만 제작 됨

### 아이템 스폰
- 아이템 제작중
- 식판 : 체력의 15% 힐
- 초록음료 : 일정 시간 무적 이후 n 초 만큼 느려짐
- exp 잼 or 스톤 or (?) : 아직 어떤 식으로 구현 할지 생각중 아마 소중대 사이즈로 나올 예정
